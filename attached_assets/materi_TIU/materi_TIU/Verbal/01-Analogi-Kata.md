# Analogi Kata

> **Bagian dari**: Tes Intelegensi Umum (TIU) — Kemampuan Verbal  
> **Tingkat kesulitan**: ⭐⭐ (mudah-sedang, sering muncul)

---

## 📖 Apa itu Analogi Kata?

**Analogi kata** menguji kemampuan Anda mengenali **hubungan logis** antara dua kata, lalu mengaplikasikan hubungan yang sama ke pasangan kata lain.

Format soal:
```
A : B = C : ...
```
artinya: hubungan A→B harus sama dengan hubungan C→jawaban.

Di SKD CPNS, biasanya muncul **3–5 soal analogi** dari ~30 soal TIU.

---

## 🎯 12 Jenis Hubungan Analogi yang Wajib Dikenali

### 1. Sinonim (kata sama makna)

```
GEMBIRA : BAHAGIA = SEDIH : DUKA
```
Hubungan: A bersinonim dengan B.

### 2. Antonim (kata lawan makna)

```
PANAS : DINGIN = TINGGI : RENDAH
```

### 3. Profesi → Alat / Tempat Kerja

```
DOKTER : STETOSKOP = TUKANG KAYU : PALU
DOKTER : RUMAH SAKIT = GURU : SEKOLAH
```

### 4. Pelaku → Hasil Karya

```
PENGARANG : BUKU = PELUKIS : LUKISAN
KOMPOSER : LAGU = ARSITEK : BANGUNAN
```

### 5. Bahan Baku → Produk

```
PADI : BERAS = TEBU : GULA
KAPAS : BENANG = SUSU : KEJU
```

### 6. Sebab → Akibat

```
HUJAN : BANJIR = API : KEBAKARAN
LELAH : TIDUR = LAPAR : MAKAN
```

### 7. Benda → Bagian / Komponen

```
MOBIL : RODA = POHON : DAUN
KOMPUTER : KEYBOARD = SEPEDA : SADEL
```

### 8. Kelompok → Anggota

```
DOKTER : PROFESI = MAWAR : BUNGA
INDONESIA : ASEAN = JERMAN : UNI EROPA
```

### 9. Satuan → Ukuran

```
JAM : MENIT = METER : CENTIMETER
KILOGRAM : GRAM = RUPIAH : SEN
```

### 10. Mata Uang → Negara

```
DOLAR : AMERIKA = YEN : JEPANG
EURO : EROPA = RUPEE : INDIA
```

### 11. Hewan → Tempat Hidup / Makanan

```
IKAN : LAUT = BURUNG : SARANG
SAPI : RUMPUT = HARIMAU : DAGING
```

### 12. Penyakit → Bagian Tubuh / Aktivitas

```
INSOMNIA : TIDUR = ANOREKSIA : MAKAN
KATARAK : MATA = DIABETES : GULA
```

## ⚡ Strategi Cepat Mengerjakan Analogi

### Langkah 1: BUAT KALIMAT HUBUNGAN

Jangan langsung lihat opsi. Tulis dulu hubungan A:B dalam kalimat singkat.

**Contoh:**  
`DOKTER : STETOSKOP = ... : ...`

Kalimat: "Dokter MENGGUNAKAN stetoskop sebagai alat utama bekerja."

### Langkah 2: TEST KALIMAT KE OPSI

Coba terapkan kalimat ke setiap opsi:
- A. Petani : sawah → "Petani menggunakan sawah sebagai alat" ❌ (sawah=tempat)
- B. Tukang kayu : palu → "Tukang kayu menggunakan palu sebagai alat" ✓
- C. Pelukis : kuas → "Pelukis menggunakan kuas sebagai alat" ✓ (mirip!)

### Langkah 3: PERHATIKAN ARAH HUBUNGAN

Kalau ada 2 opsi yang mirip, **arah hubungan** menentukan.

Contoh: `BUKU : PENGARANG = ... : ...`
- Arah: hasil karya → pencipta
- Bukan: profesi → hasil

### Langkah 4: ELIMINASI HUBUNGAN YANG TERLALU UMUM

Hindari hubungan terlalu longgar seperti "berkaitan dengan". Cari hubungan **SPESIFIK**.

---

## 💡 Tips & Trik Penting

| Tip | Penjelasan |
|-----|-----------|
| **Buat kalimat lengkap** | Hubungan A→B harus bisa dijelaskan dalam 1 kalimat utuh |
| **Jangan terjebak makna umum** | "Mobil dan motor sama-sama kendaraan" terlalu longgar |
| **Perhatikan tingkat hubungan** | Sapi→susu beda dengan sapi→daging (produk vs hasil) |
| **Jika ragu, pilih yang paling spesifik** | Hubungan yang sangat presisi biasanya jawabannya |
| **Cek arah dari kanan kiri** | A:B harus sama arahnya dengan C:D |

## 📝 Contoh Soal & Pembahasan

### Soal 1 (Mudah)
**GURU : MENGAJAR = DOKTER : ...**

A. Sakit  
B. Rumah Sakit  
C. Mengobati  
D. Pasien  
E. Stetoskop

**Pembahasan**: Hubungan = profesi : pekerjaan inti. Guru tugasnya mengajar; dokter tugasnya mengobati.  
**Jawaban: C**

---

### Soal 2 (Sedang)
**PADI : BERAS = ... : ...**

A. Tepung : Roti  
B. Tebu : Gula  
C. Susu : Keju  
D. Sapi : Daging  
E. Buah : Jus

**Pembahasan**: Padi → diolah jadi beras (proses tunggal: pengupasan). Yang paling mirip: tebu → diolah jadi gula. A salah (tepung sudah olahan). C butuh fermentasi (banyak proses).  
**Jawaban: B**

---

### Soal 3 (Sulit — bingung dengan 2 jawaban mirip)
**JAM : MENIT = ... : ...**

A. Hari : Jam  
B. Tahun : Bulan  
C. Meter : Centimeter  
D. Kilogram : Gram  
E. Detik : Milidetik

**Pembahasan**: Hubungan = "satuan waktu lebih besar : satuan waktu lebih kecil tepat di bawahnya". 1 jam = 60 menit; 1 hari = 24 jam (juga waktu, tepat di bawah). Pilihan B, C, D bukan satuan waktu. E juga waktu tapi sub-detik (jarang dipakai).  
**Jawaban: A**

---

### Soal 4 (Sulit — hubungan tidak harfiah)
**INSOMNIA : TIDUR = ... : ...**

A. Anemia : Darah  
B. Hipertensi : Tekanan  
C. Diabetes : Gula  
D. Anoreksia : Makan  
E. Migrain : Kepala

**Pembahasan**: Hubungan = "gangguan : aktivitas/fungsi yang TERGANGGU". Insomnia = gangguan tidur (tidak bisa tidur). Anoreksia = gangguan makan (tidak mau makan).  
A=anemia berkaitan dengan darah tapi bukan "gangguan aktivitas".  
**Jawaban: D**

---

### Soal 5 (Sangat Sulit — multi-level)
**KOPI : KAFEIN = TEH : ...**

A. Daun  
B. Tanin  
C. Air  
D. Cangkir  
E. Sore

**Pembahasan**: Hubungan = "minuman : zat aktif utama". Kopi mengandung kafein; teh mengandung tanin (dan juga sedikit kafein, tapi tanin = zat khasnya).  
**Jawaban: B**

## ⚠️ Kesalahan Umum

1. **Terjebak makna kata pertama saja**  
   Soal mengandalkan HUBUNGAN, bukan arti masing-masing kata.

2. **Memilih hubungan terlalu longgar**  
   "Sama-sama berkaitan dengan X" bukan jawaban analogi yang baik.

3. **Tidak memperhatikan arah**  
   A:B beda dengan B:A. "Buku:Pengarang" ≠ "Pengarang:Buku".

4. **Ngotot dengan jawaban pertama yang terlihat benar**  
   Selalu cek SEMUA opsi sebelum memilih — sering jawaban A "sepertinya benar" padahal C lebih tepat.

5. **Lupa cek konsistensi**  
   Pastikan hubungan A:B = C:D bisa dijelaskan dengan KALIMAT YANG SAMA.

---

## 🎯 Target Kemampuan

- **Mudah** (sinonim, antonim): < 10 detik
- **Sedang** (profesi, produk, fungsi): < 20 detik
- **Sulit** (hubungan abstrak/multi-level): < 30 detik

**Target benar**: minimal **3 dari 4** soal analogi.
