# Penalaran Analitis

> **Bagian dari**: Tes Intelegensi Umum (TIU) — Kemampuan Verbal  
> **Tingkat kesulitan**: ⭐⭐⭐⭐ (sulit, butuh konsentrasi tinggi)

---

## 📖 Apa itu Penalaran Analitis?

**Penalaran analitis** menguji kemampuan Anda menarik kesimpulan dari **sekumpulan informasi/aturan** untuk:
- Menentukan urutan (siapa di mana, kapan, urutan ke berapa)
- Mencocokkan elemen (siapa punya apa)
- Menyimpulkan hubungan logis

Di SKD CPNS, biasanya muncul **3–5 soal** dari ~30 soal TIU.

⚠️ **Catatan**: ini soal yang paling banyak makan waktu. Strategi yang benar = **selesai cepat, atau skip dulu untuk dikerjakan terakhir**.

---

## 🎯 Jenis-Jenis Soal Penalaran Analitis

### 1. Urutan / Ranking

**Contoh tipikal:**
> Tinggi Adi > Budi > Cici. Dedi = Adi. Eni < Cici. Siapa paling pendek?

**Pendekatan:**
1. Tulis informasi sebagai rantai pertidaksamaan
2. Susun dari terbesar ke terkecil

```
Adi = Dedi  >  Budi  >  Cici  >  Eni
```

→ Paling pendek: **Eni**

---

### 2. Posisi / Tempat Duduk

**Contoh tipikal:**
> A, B, C, D, E duduk berjajar. A di ujung kiri. C di kanan B. D di antara C dan E. Urutkan!

**Pendekatan:**
1. Tetapkan posisi tetap (A di kiri = pos 1)
2. Susun fragmen: B-C berdekatan (B kiri C), D-E dengan D di tengah → C-D-E
3. Gabung: A-B-C-D-E

---

### 3. Pencocokan (Matching)

**Contoh tipikal:**
> Ana, Bita, Cana, Dina punya hewan: kucing, anjing, kelinci, ikan. Ana bukan kucing/kelinci. Bita = anjing. Cana bukan ikan. Hewan Dina?

**Pendekatan: gunakan TABEL eliminasi**

|  | Kucing | Anjing | Kelinci | Ikan |
|---|---|---|---|---|
| Ana | ❌ | (Bita) | ❌ | ✓ |
| Bita | | ✓ | | |
| Cana | ✓ | | | ❌ |
| Dina | | | ✓ | |

→ Dina = **kelinci**

---

### 4. Penjadwalan

**Contoh tipikal:**
> Tim main 4 hari: A, B, C, D. A main sebelum B. C main setelah D. D main hari pertama. Urutkan!

**Pendekatan:**
1. D pasti hari 1
2. C setelah D
3. A sebelum B → A bisa di mana saja sebelum B

Kemungkinan urutan: D-A-C-B atau D-C-A-B

---

### 5. Penalaran Kombinasi Aturan

**Contoh tipikal:**
> 5 mobil di parkiran: putih, merah, hitam, biru, hijau. Putih di antara hitam dan merah. Hijau paling kanan. Biru di kiri hitam.

**Pendekatan:**
1. Hijau = pos 5 (fixed)
2. Biru-Hitam = berdekatan, Biru di kiri
3. Hitam-Putih-Merah atau Merah-Putih-Hitam (3 berturut-turut)
4. Coba kombinasi: Biru-Hitam-Putih-Merah-Hijau ✓

---

## ⚡ Strategi Cepat

### Langkah 1: BACA SOAL SAMPAI HABIS

Jangan langsung kerja. Pahami siapa, di mana, kapan, berapa.

### Langkah 2: TULIS SEMUA ATURAN dengan SIMBOL

Contoh: "A lebih tua dari B" → `A > B` (di kertas corat-coret).

### Langkah 3: TEMUKAN ATURAN PALING SPESIFIK

Mulai dari aturan yang menentukan posisi tetap (fixed point).
- "D di hari pertama" → fixed di pos 1
- "Hijau paling kanan" → fixed di pos 5
- "X anak bungsu" → fixed di posisi terakhir

### Langkah 4: GUNAKAN TABEL untuk SOAL PENCOCOKAN

Tabel = cara paling efisien. Hindari mengingat semua di kepala.

### Langkah 5: ELIMINASI OPSI

Kalau buntu, coba masukkan opsi ke aturan. Mana yang BERTENTANGAN dengan aturan = bukan jawaban.

---

## 💡 Tips & Trik Penting

| Tip | Penjelasan |
|-----|-----------|
| **Selalu corat-coret di kertas** | Jangan ditahan di kepala |
| **Gunakan simbol/diagram** | Lebih cepat dari kata-kata |
| **Mulai dari aturan paling kaku** | "X tepat di posisi N" lebih kuat dari "X sebelum Y" |
| **Tabel = teman terbaik** | Untuk soal matching/pencocokan |
| **Verifikasi semua aturan** | Setelah ketemu, cek lagi—pastikan aturan terpenuhi |
| **Jangan terjebak waktu** | Kalau 2 menit belum dapat, skip dan kerjakan terakhir |

## 📝 Contoh Soal & Pembahasan

### Soal 1 (Mudah — Urutan)
**Andi lebih tua dari Budi. Budi lebih tua dari Cici. Cici lebih tua dari Dedi. Siapa termuda?**

A. Andi  B. Budi  C. Cici  D. Dedi  E. Tidak dapat ditentukan

**Pembahasan:**  
Andi > Budi > Cici > Dedi → termuda **Dedi**.  
**Jawaban: D**

---

### Soal 2 (Sedang — Posisi Duduk)
**A, B, C, D, E duduk berjajar. A di ujung kiri. C di kanan B. D di antara C dan E. Urutan dari kiri ke kanan?**

A. A-B-C-D-E  
B. A-C-B-D-E  
C. A-B-C-E-D  
D. A-D-B-C-E  
E. A-E-B-C-D

**Pembahasan:**  
- A di pos 1 (fixed)  
- C kanan B → ...B-C...  
- D di antara C dan E → ...C-D-E (D di tengah)  
- Gabung: A-B-C-D-E  
**Jawaban: A**

---

### Soal 3 (Sedang — Matching)
**Ana, Bita, Cana, Dina punya hewan berbeda: kucing, anjing, kelinci, ikan.**  
**Ana tidak punya kucing/kelinci. Bita punya anjing. Cana tidak punya ikan. Hewan Dina adalah?**

A. Kucing  B. Anjing  C. Kelinci  D. Ikan  E. Tak ditentukan

**Pembahasan (pakai tabel):**

|  | Kucing | Anjing | Kelinci | Ikan |
|---|---|---|---|---|
| Ana | ❌ | ❌ (Bita) | ❌ | **✓** |
| Bita | | **✓** | | |
| Cana | **✓** | | | ❌ |
| Dina | | | **✓** | |

- Bita = anjing (langsung dari soal)
- Ana bukan kucing/kelinci/anjing → Ana = ikan
- Cana bukan ikan → Cana = kucing (sisa kelinci & kucing)
- Dina = kelinci

**Jawaban: C**

---

### Soal 4 (Sulit — Multi Aturan)
**Lima mobil di parkiran: putih, merah, hitam, biru, hijau. Putih di antara hitam dan merah. Hijau paling kanan. Biru di kiri hitam.**

A. Biru-Hitam-Putih-Merah-Hijau  
B. Hitam-Putih-Merah-Biru-Hijau  
C. Biru-Merah-Putih-Hitam-Hijau  
D. Merah-Putih-Hitam-Biru-Hijau  
E. Hijau-Biru-Hitam-Putih-Merah

**Pembahasan:**  
- Hijau di pos 5 (fixed) → opsi E gugur  
- Biru di kiri Hitam, mereka berdekatan → Biru-Hitam → opsi B, D gugur (Biru bukan langsung kiri Hitam)  
- Putih di antara hitam-merah: blok "Hitam-Putih-Merah" atau "Merah-Putih-Hitam"  
- Kombinasi dengan Biru-Hitam: Biru-Hitam-Putih-Merah-Hijau ✓  
**Jawaban: A**

---

### Soal 5 (Sangat Sulit — Logika Posisi)
**Dalam rapat, A duduk berhadapan dengan B. C di kanan A. D di kanan B. Siapa berhadapan dengan C?**

A. A  B. B  C. D  D. Tidak ada  E. Tak ditentukan

**Pembahasan:**  
Bayangkan meja: A di utara, B di selatan (berhadapan).  
- "Kanan A" = arah dari sudut A → barat (jika A menghadap selatan ke B)  
- "Kanan B" = arah dari sudut B → timur (jika B menghadap utara ke A)  

Hmm... ini tergantung interpretasi. Lebih sederhana: kalau A-B berhadapan, dan C, D di kiri-kanannya, maka C dan D pasti berhadapan satu sama lain.

**Jawaban: C** (D berhadapan dengan C)

---

## ⚠️ Kesalahan Umum

1. **Tidak corat-coret**  
   Mencoba mengingat 5 aturan di kepala = gagal pasti.

2. **Tidak mulai dari aturan terkuat**  
   Mulai dari "X di posisi tetap" lalu kembangkan.

3. **Tidak verifikasi**  
   Setelah ketemu jawaban, cek lagi—semua aturan harus terpenuhi.

4. **Bingung "kiri-kanan" relatif**  
   "Sebelah kanan X" dari sudut pandang X, bukan kita yang lihat dari depan.

5. **Terlalu lama di satu soal**  
   Kalau 2 menit buntu, skip. Kerjakan soal lain dulu, balik lagi nanti.

---

## 🎯 Target Kemampuan

- **Mudah** (urutan sederhana): < 45 detik
- **Sedang** (posisi, matching): < 90 detik
- **Sulit** (multi-aturan kombinasi): < 120 detik

**Target benar**: minimal **2 dari 4** soal analitis (paling sulit, jangan kebablasan).

## 🔁 Latihan Mandiri

1. **A, B, C, D mengantri. C tepat di belakang A. B paling depan. Siapa di posisi ke-3?**

2. **Tinggi Pak Andi > Pak Budi. Pak Cici < Pak Dedi. Pak Budi = Pak Dedi. Urutkan dari tertinggi.**

3. **Tiga teman (P, Q, R) memilih warna baju: merah, biru, hijau. P tidak merah. Q tidak biru. R = biru. Warna baju P?**

4. **Dalam lomba: Andi finis sebelum Budi. Cici finis setelah Dedi. Dedi sebelum Andi. Eni terakhir. Siapa juara 1?**

5. **5 buku ditumpuk. Buku merah di paling atas. Buku biru di antara hijau dan kuning. Putih di paling bawah. Hijau di atas biru. Urutkan dari atas ke bawah.**

**Kunci jawaban:**  
1. D  
2. Andi > Budi=Dedi > Cici  
3. Hijau (P bukan merah, R=biru, sisa merah=Q, P=hijau)  
4. Dedi  
5. Merah-Hijau-Biru-Kuning-Putih
