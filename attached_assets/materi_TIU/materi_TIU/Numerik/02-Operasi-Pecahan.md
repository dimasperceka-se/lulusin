# Operasi Pecahan

> **Bagian dari**: TIU — Kemampuan Numerik  
> **Tingkat kesulitan**: ⭐⭐ (mudah-sedang)

---

## 📖 Apa itu Operasi Pecahan?

Pecahan = bagian dari keseluruhan, ditulis a/b (a = pembilang, b = penyebut).

Soal pecahan di TIU mencakup:
- Penjumlahan, pengurangan, perkalian, pembagian pecahan
- Pecahan campuran
- Konversi pecahan ↔ desimal ↔ persen

Biasanya muncul **2–4 soal** di SKD CPNS.

---

## 🎯 Konsep Dasar yang Wajib Dikuasai

### 1. Pecahan Senilai

Mengalikan/membagi pembilang & penyebut dengan angka yang sama:

```
1/2 = 2/4 = 3/6 = 4/8 = 5/10
```

### 2. Penyederhanaan

Bagi pembilang & penyebut dengan FPB:

```
12/18 = 2/3   (FPB = 6)
24/36 = 2/3   (FPB = 12)
```

### 3. Pecahan Campuran ↔ Biasa

```
Pecahan campuran → biasa:
2 ⅓ = (2×3 + 1) / 3 = 7/3

Pecahan biasa → campuran:
7/3 = 2 sisa 1 → 2⅓
```

---

## ➕ Operasi Penjumlahan & Pengurangan

### Penyebut Sama
Langsung jumlahkan/kurangkan pembilangnya:
```
3/7 + 2/7 = 5/7
5/9 − 1/9 = 4/9
```

### Penyebut Berbeda (samakan dulu!)
Pakai KPK penyebut.

```
1/2 + 1/3 = ?

KPK 2 dan 3 = 6
→ 3/6 + 2/6 = 5/6
```

**Trik silang (untuk 2 pecahan):**
```
a/b + c/d = (a×d + c×b) / (b×d)

Contoh: 2/3 + 3/4 = (2×4 + 3×3)/(3×4) = (8+9)/12 = 17/12
```

---

## ✖️ Operasi Perkalian & Pembagian

### Perkalian
Pembilang × pembilang, penyebut × penyebut:

```
2/3 × 4/5 = (2×4)/(3×5) = 8/15

Sederhanakan dulu kalau bisa:
2/3 × 9/4 = (2×9)/(3×4) 
          = 18/12 = 3/2
          atau lebih cepat: silang sederhanakan
          = (1×3)/(1×2) = 3/2
```

### Pembagian
Kalikan dengan kebalikan:

```
a/b ÷ c/d = a/b × d/c

Contoh: 2/3 ÷ 4/5 = 2/3 × 5/4 = 10/12 = 5/6
```

---

## 🔄 Konversi Pecahan ↔ Desimal ↔ Persen

### Pecahan → Desimal
Bagi pembilang dengan penyebut:
```
3/4 = 3 ÷ 4 = 0,75
1/8 = 0,125
2/5 = 0,4
```

### Desimal → Pecahan
```
0,5 = 5/10 = 1/2
0,75 = 75/100 = 3/4
0,125 = 125/1000 = 1/8
```

### Pecahan ↔ Persen
- × 100% untuk jadi persen: 3/4 = 75%
- ÷ 100% untuk balik: 60% = 60/100 = 3/5

## ⚡ Strategi Cepat

### Langkah 1: SAMAKAN BENTUK

Kalau soal campur (pecahan + desimal + persen), ubah semuanya jadi 1 bentuk (biasanya pecahan).

### Langkah 2: SEDERHANAKAN DULU SEBELUM HITUNG

Contoh: 6/9 × 12/8
- Jangan langsung kali. Sederhanakan dulu: 6/9 = 2/3; 12/8 = 3/2
- 2/3 × 3/2 = 1 ✓

### Langkah 3: GUNAKAN KPK YANG TEPAT

KPK terkecil = hitungan lebih cepat. Untuk 2 angka coprime (tidak punya faktor sama), KPK = hasil kali.

### Langkah 4: PROSES OPERASI BERTINGKAT

Ikuti urutan:
1. Kurung
2. Perkalian/pembagian (kiri ke kanan)
3. Penjumlahan/pengurangan (kiri ke kanan)

## 💡 Tips & Trik

| Tip | Penjelasan |
|-----|-----------|
| **Hafal pecahan-desimal-persen** | Hemat waktu konversi |
| **Sederhanakan SEBELUM operasi** | Angka kecil = hitung cepat |
| **Untuk pembagian: balikkan & kali** | a÷b = a × 1/b |
| **Trik silang untuk 2 pecahan** | (a/b)+(c/d) = (ad+cb)/bd |
| **Cek hasil**: hasil > 1 atau < 1? | Pecahan biasa ÷ pecahan biasa hasilnya bisa > 1 |

## 📝 Contoh Soal & Pembahasan

### Soal 1 (Mudah)
**¾ + ⅔ − ½ = ?**

**Pembahasan:**
- KPK 4, 3, 2 = 12
- ¾ = 9/12; ⅔ = 8/12; ½ = 6/12
- 9/12 + 8/12 − 6/12 = 11/12

**Jawaban: 11/12**

---

### Soal 2 (Sedang)
**2 ⅓ + 1 ½ = ?**

**Pembahasan:**
- 2⅓ = 7/3; 1½ = 3/2
- KPK 3 dan 2 = 6
- 7/3 = 14/6; 3/2 = 9/6
- 14/6 + 9/6 = 23/6 = 3 5/6

**Jawaban: 3 5/6 atau 23/6**

---

### Soal 3 (Sedang)
**¾ × 2/9 = ?**

**Pembahasan:**
- ¾ × 2/9 = (3×2)/(4×9) = 6/36 = 1/6
- Atau sederhanakan dulu: 3/4 × 2/9 = (3/9)×(2/4) = 1/3 × 1/2 = 1/6

**Jawaban: 1/6**

---

### Soal 4 (Sulit)
**(5/8) : (3/4) = ?**

**Pembahasan:**
- Pembagian = kali kebalikan
- 5/8 ÷ 3/4 = 5/8 × 4/3 = 20/24 = 5/6

**Jawaban: 5/6**

---

### Soal 5 (Sulit — Operasi campur)
**Hasil dari 5/8 dari 64 dikurangi 3/4 dari 32 adalah?**

**Pembahasan:**
- 5/8 × 64 = 5 × 8 = 40
- 3/4 × 32 = 3 × 8 = 24
- 40 − 24 = **16**

**Jawaban: 16**

---

## ⚠️ Kesalahan Umum

1. **Lupa samakan penyebut**  
   `1/2 + 1/3 ≠ 2/5`! Harus disamakan dulu.

2. **Salah balik pembagian**  
   `a/b ÷ c/d ≠ a/b × c/d` — yang benar `× d/c`.

3. **Tidak sederhanakan dulu**  
   Hitung dengan angka besar boros waktu & risiko salah.

4. **Pecahan campuran salah konversi**  
   `2 ⅓ = 7/3`, bukan `2/3 + 1/3`.

5. **Salah baca pecahan campuran**  
   Pastikan tahu mana pembilang & penyebut.
